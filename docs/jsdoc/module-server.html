Module: server

MuzaLife backend server entry point.

Configures and starts an HTTPS Express server. All route modules are mounted here and a Swagger UI is exposed at /api/docs for interactive API testing.

Architecture overview:

  • All routes are prefixed with /api/.
  • Authentication is enforced per-route via the authenticateToken middleware (see middleware/auth.js).
  • Static product files are served from the uploads/ directory.
  • HTTPS is mandatory even in development; certificates must exist in certs/ before the server can start.
