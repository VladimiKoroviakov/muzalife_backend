Module: middleware/performanceMonitor

Performance monitoring middleware for MuzaLife backend.

Collects per-endpoint latency metrics, aggregates them in-memory, and exposes the summary via the APM stats endpoint (GET /api/apm/stats).

Architecture: metrics are stored in a plain JS Map keyed by "METHOD /route". Each bucket holds a rolling window of the last MAX_SAMPLES latency samples plus derived statistics (min, max, p50, p95, p99, avg, count). No external dependency is required.

Integration: mount this middleware before route handlers so the full handler latency is captured.
