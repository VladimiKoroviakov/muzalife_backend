Class: EmailService

services/emailService~EmailService()

Service responsible for sending transactional emails via SMTP.

Constructor

new EmailService()

Initialises the service and creates the underlying nodemailer transporter.

The constructor intentionally does not throw on SMTP configuration errors (e.g. missing env vars) — it only logs warnings so that other server functionality remains available.

extractVerificationCode(html) → {string}

Extracts a 6-digit numeric verification code from an HTML email body.

Used primarily in integration tests to retrieve the code from the rendered HTML without actually receiving the email.

Algorithm: searches for the first occurrence of >NNNNNN< where N is a digit (matches the code rendered inside an HTML tag).

(async) sendVerificationEmail(email, verificationCode, verification_typeopt) → {Promise.<true>}

Sends a verification email containing a one-time code to the given address.

The email is bilingual-ready but currently rendered in Ukrainian. The subject line and body copy differ slightly between 'registration' and 'email_change' flows.
