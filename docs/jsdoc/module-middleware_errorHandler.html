Module: middleware/errorHandler

Global error-handling middleware for MuzaLife Express server.

Responsibilities:

  1. Classifies errors as operational (AppError) or programming bugs.
  2. Logs every error with full context: errorId, requestId, userId, stack, duration, and any custom context attached to the error.
  3. Sends a sanitised JSON response — no internal stack traces leak to clients.
  4. Returns user-facing messages in both Ukrainian (uk) and English (en) via the errorMessages localisation map.

This middleware MUST be registered as the last app.use() call in server.js.
