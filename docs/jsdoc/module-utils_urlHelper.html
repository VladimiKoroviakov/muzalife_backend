Module: utils/urlHelper

URL construction helpers for the MuzaLife backend.

Utility functions for normalising resource URLs. Relative paths stored in the database (e.g. /uploads/products/image.jpg) are converted to absolute URLs before they are sent to the client, so the frontend never needs to know the server hostname.

(static, constant) constructFullUrl

Builds a fully-qualified URL from a (possibly relative) image path.

If imagePath is already absolute (http://… or https://…) it is returned unchanged. Otherwise the protocol and host are taken from the current Express request object and prepended to the path.

Why this exists: product images are stored as relative paths in the database so the server can be deployed to any domain without a data migration. This helper is called in every controller that returns image URLs to the client.

Examples
// Relative path → absolute URL
constructFullUrl(req, '/uploads/products/42/cover.jpg');
// => 'https://localhost:5001/uploads/products/42/cover.jpg'
// Already-absolute URL passed through unchanged
constructFullUrl(req, 'https://cdn.example.com/image.png');
// => 'https://cdn.example.com/image.png'
// Null/undefined → null
constructFullUrl(req, null);
// => null
